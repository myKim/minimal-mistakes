# 아이템 7 : 인스턴스 변수에 내부에서 접근할 때는 직접 접근하라

## 요약

- 내부에서는 인스턴스 변수를 읽을 때(read)는 직접 접근해 읽고, 반대로 쓸 때(write)는 프로퍼티를 이용하라.
- 초기화 메서드와 dealloc에서는 항상 인스턴스 변수를 직접 읽고 쓰라.
- 데이터(즉 인스턴스 변수)가 지연 초기화를 이용한다면 프로퍼티를 이용해 데이터를 읽어야 한다.
